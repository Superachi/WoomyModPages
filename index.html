Welcome to the WoomyMod Wiki!

Here you can find information about the WoomyMod, a fan-made mod for Terraria that adds Splatoon-related content to the game.
This is my first mod project and "website", so please bear with me :)
This is a non-profit project, and is unaffiliated with Nintendo. Splatoon belongs to them.

Please note: most of the information was generated via code in-game, so it may be incomplete or incorrect. Also, unlike a regular wiki, you can't make edits, as this is a static site (using only HTML/CSS, there's no backend to save changes to).
