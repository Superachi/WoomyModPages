Welcome to the WoomyMod Wiki!

Here you can find information about the WoomyMod, a fan-made, singleplayer mod for Terraria that adds Splatoon-related items to the game.
This is a non-profit project, and is unaffiliated with Nintendo. Splatoon belongs to them.

Please note: this wiki is not yet complete, more information will be added when I find the time. I'm also generating the shown data based on the source code, so some things may be missing/inaccurate.

I hope you'll enjoy the mod!
